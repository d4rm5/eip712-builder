# EIP-712 sign buidler

Scaffold ETH 2 extension to build and test [EIP-712](https://eips.ethereum.org/EIPS/eip-712) signatures easy.

## Installation

```bash
npx create-eth@latest -e d4rm5/eip712-builder
```
